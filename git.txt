mkdir test - создать каталог test
cd test - войти в test
git init - создать репозиторий
git status - состояние репозитория,есть ли изменения - за чем он наблюдает....
dir - содержимое текущего каталога
git add имя_файла- добавляем файл(закоммитим)
git commit -m "текст комментария"
git log - смотрим состояние коммитов-фиксированных изменений - история коммитов(слепков)


git config --global user.name 'Anna Mahovskaja'
git config --global user.email 'mahovskie@mail.ru'

git config - смотреть команды

если перед соммитом забыли добавить файл в индекс с помощью add
 можно использовать git commit -am "текст комментария"

git diff  - посмотреть какие изменения произошли
